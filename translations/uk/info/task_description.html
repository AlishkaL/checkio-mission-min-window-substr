За заданим рядком line та шаблоном pattern напишіть функцію, яка знаходить мінімальний підрядок вікна line, який містить принаймні всі символи з pattern (можуть бути й інші символи). Порядок символів у рядку line/pattern не має значення, але має значення регістр літер.

Ваша функція повинна повернути кортеж початкового та кінцевого індексів line, які формують потрібне вікно (права межа не враховується). Якщо у line немає такого підрядка, який би відповідав pattern, поверніть {% if is_js %}[-1, -1]{% else %}(-1, -1){% endif %}.

Вхідні дані: Два рядки (str{% if is_js %}ing{% endif %}).

Вихідні дані: Кортеж з двох цілих чисел ({% if is_js %}number{% else %}int{% endif %}).
