Biorąc pod uwagę ciąg line i wzorzec pattern, proszę napisać funkcję, która znajdzie minimalny podciąg okna line, który zawiera co najmniej wszystkie znaki z pattern (może zawierać również inne znaki). Kolejność znaków w line/pattern nie ma znaczenia, ale wielkość liter ma.

Państwa funkcja powinna zwrócić krotkę początkowych i końcowych indeksów line, które tworzą wymagane okno (prawe obramowanie jest wykluczone). Jeśli nie ma takiego podciągu w line, który pasowałby do pattern, proszę zwrócić {% if is_js %}[-1, -1]{% else %}(-1, -1){% endif %}.

Dane wejściowe: Dwa ciągi znaków (str{% if is_js %}ing{% endif %}).

Wyjście: Krotka dwóch liczb całkowitych ({% if is_js %}number{% else %}int{% endif %}).
